# BiKatıl Mobile

`BiKatıl`, kullanıcıların şehirlerindeki etkinlikleri keşfedebildiği, katılım sağlayabildiği ve organizer tarafında kendi etkinliklerini yönetebildiği mobil uygulamadır.

## Geliştirme

1. Bağımlılıkları kur:

```bash
npm install
```

2. Geliştirme sunucusunu başlat:

```bash
npm run start
```

3. Platformlardan birini aç:

- `npm run android`
- `npm run ios`
- `npm run web`

## Yapı

- `src/app`: Expo Router tabanlı ekranlar
- `src/components`: Ortak arayüz bileşenleri ve dialog sistemi
- `src/services`: API ve kimlik doğrulama yardımcıları
- `src/constants`: Tema, renk ve tipografi tanımları

## Marka Notu

Uygulamanın görünen adı `BiKatıl` olarak kullanılır. Mobil kabuk, onboarding ve keşif dili bu marka etrafında kurgulanmıştır.

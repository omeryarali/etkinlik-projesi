Projenin amacı:

Bölgesel etkinliklerin ve turnuvaların listelendiği, kullanıcıların etkinliklere katılabildiği, organizatörlerin etkinlik oluşturabildiği mobil uygulama.



Roller:

\- Katılımcı

\- Organizatör

\- Admin



Temel akış:

\- Katılımcı etkinlikleri görür ve katılır.

\- Katılımcı isterse organizatör başvurusu yapar.

\- Admin organizatör başvurusunu onaylar.

\- Onaylanan organizatör etkinlik oluşturabilir.

\- Oluşturulan etkinlik admin onayından sonra yayına çıkar.
